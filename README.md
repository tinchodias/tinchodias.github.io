This is my personal site. 

It is generated and deployed with (Ecstatic)[https://guillep.github.io/ecstatic/].

# How to use

1. Clone "source" branch.
2. Run "ecstatic generate" to generate from pillar files and templates.
3. Test with "ecstatic serve".
4. Deploy "ECSTATIC_DIR/_scripts/publish-github-io.sh"
