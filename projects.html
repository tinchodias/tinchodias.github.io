Projects

I participate in several open source software projects.

Pharo

I'm involved in the Pharo open-source community. Pharo is a pure object-oriented programming language and a powerful environment, focused on simplicity and immediate feedback. Pharo is a Smalltalk-inspired language that offers strong live programming features such as immediate object manipulation, live update, hot recompilation. Live programming environment is in the heart of the system.

Fuel

I've worked in the development, and currently in maintenance tasks, of Fuel. Fuel is a general-purpose object serialization framework developed in Pharo.

Epicea

I'm working in Epicea. Epicea is a tool for Pharo that records code changes enriched with IDE events. Epicea extends and could replace the traditional .changes logging mechanism, where the recorded code changes are incomplete and are not properly reified and thus it can be difficult to recover lost code after an image crash.
